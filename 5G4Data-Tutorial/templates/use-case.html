5G4Data Use Case

5G4Data is Telenor's use case within the INTEND project and it explores how to make advanced 5G network and compute capabilities accessible through intent-driven automation. It aligns with industry efforts, particularly TM Forum’s work on Autonomous Networks, and leverages their standardized intent models to interpret customer goals.

The use case demonstrates how natural language input from users can drive automated network configuration—such as dynamic slice provisioning—and application deployment to Edge data centres. This approach enables low-latency, high-performance applications, particularly where centralized cloud solutions are insufficient.

By deploying containerized workloads closer to the data source, response times are minimized, ensuring performance for time-sensitive applications. The system must determine whether the user's intent requires network configuration, Edge deployment, or a combination of both.

Though focused in scope, the project supports a larger vision of autonomous, intent-based network operations—crucial for managing increasing complexity and meeting future service demands. As networks become more complex and customer expectations continue to rise, this shift will be critical. The INTEND project is an important step in Telenor's journey toward meeting those challenges and preparing for the next generation of communication services.
