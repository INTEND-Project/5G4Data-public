How Quick Populate Works

When you select the "Select intents and quick populate" option, the system will automatically generate observation reports with randomly selected parameters:

Random Mode Selection

The system randomly selects one of these generation modes:

  • random: Uniform random values
  • diurnal: Daily sinusoidal pattern
  • walk: Mean-reverting random walk
  • trend: Linear trend with noise
Random Anomaly Settings

If anomalies are enabled, random parameters are selected:

  • Strategy: random, fixed, or peak
  • Rate: 0.01-0.05 for random/peak
  • Interval: 1h-12h for fixed
  • Duration: 2-5 samples
  • Amplitude: 20-50% of value range
  • Direction: spike, dip, or both
  • Peak hours: 14-22 UTC
Smart Value Range Detection

The system automatically deduces min/max values from the quan operators in your intent conditions (e.g., quan:atLeast: 500mbit/s). This ensures generated values are contextually appropriate for each condition.
